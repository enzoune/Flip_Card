# 💳 Interactive Credit Card UI (Interface de Cartão Realista)

> Um componente de interface sofisticado que recria a estrutura visual de um cartão de crédito, focado em realismo, semântica e organização de layout.

## 📖 Sobre o Projeto

Este projeto integra o meu portfólio de estudos em **Programação Full-Stack**. O objetivo foi construir um **Cartão que Gira** (ou de face dupla), focando na disposição precisa de elementos complexos como chips, logotipos e dados de segurança em um layout fixo e estilizado.

O projeto demonstra a capacidade de transformar dados estruturados em uma representação visual fiel ao mundo real, utilizando apenas tecnologias web fundamentais.

## ✨ Destaques e Diferenciais Técnicos

* **Arquitetura de Faces Duplas:** Implementação de classes distintas para a frente (`front-face`) e o verso (`back-face`) do cartão, permitindo o gerenciamento independente de cada estado visual.
* **Estruturação Semântica:** Uso de tags como `<header>` para a marca do cartão e `section` para o encapsulamento do componente, garantindo um código organizado e profissional.
* **Gestão de Ativos Visuais:** Integração de imagens para elementos críticos de identidade visual (logo e chip) com atributos de acessibilidade devidamente preparados.
* **Organização de Dados:** Disposição clara de informações dinâmicas simuladas, como o número do cartão, nome do portador e data de validade, utilizando uma hierarquia de títulos (`h5`, `h6`) para clareza visual.
* **SEO e Metadados:** Inclusão de meta tags de autor e compatibilidade, demonstrando atenção aos padrões de desenvolvimento modernos e documentação de autoria.
* **Fidelidade Visual:** Recriação de elementos realistas no verso, como a faixa magnética (`magnetic-strip`) e o campo de assinatura.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação avançada e marcação de dados.
* **CSS3:** Gerenciamento de layout, tipografia e design de componentes espelhados.
